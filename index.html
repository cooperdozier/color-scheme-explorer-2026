Color Scheme Explorer

A tool for exploring and choosing color schemes with schemes generated by thecolorapi.com and manual controls, the ability to save schemes and convert them or single colors into different color spaces. And the ability to test them against preset and custom backgrounds. Once you've explored and gotten some an idea of what colors you want in your project try opening one tab where you build the exact colors in one by one.

On each page the stuff that actually makes it work is shown by clicking on the "Controls" button, the color and data display area is below, but is initially blank. Try clicking on 'Circles', then 'Controls', then use the color panel selector to tell the API to generate a scheme, use the backgrounds dropdown to test it against different backgrounds, try using the individual color selector panels to alter it, and save it for display on the history and color conveerter pages.

Inputs with the word 'exact' in them take six digit hexadecimal numbers (0-9 plus a-f, case insensitive).
